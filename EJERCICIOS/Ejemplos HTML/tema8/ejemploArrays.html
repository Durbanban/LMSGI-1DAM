EJEMPLO CON ARRAYS

.concat() encadena dos arrays.

.pop() selecciona el último elemento de un array y lo saca.

.push() mete un elemento al final del array.

Siempre es bueno usar pop y push para quitar y poner elementos a un array.

.shift() selecciona el primer elemento de un array y lo saca.

.unshift() mete un elemento al inicio del array.

.reverse() coge un array e invierte el orden de los elementos.

.splice(inicio posición, cuantos argumentos quitas, cuantos argumentos añades).

.indexOf() devuelve el primer índice para el objeto que se le pase.

.lastIndexOf() devuelve el último índice para el objeto que se le pase.
